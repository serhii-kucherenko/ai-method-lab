# AI Governance Desk

Conjoint preference scoring desk for AI governance proposals — preference-aligned vs tech-first baselines.

**Paper:** https://arxiv.org/abs/2607.14585v1  
**Authors' code:** none published (OSF: https://osf.io/5rz9p/)  
**Guide:** [docs/guides/23-ai-governance-desk-lessons.md](../../docs/guides/23-ai-governance-desk-lessons.md)  
**Design:** [docs/ideas/ai-governance-desk-DESIGN.md](../../docs/ideas/ai-governance-desk-DESIGN.md)

## Run

```bash
npm install
npm run dev          # Next.js UI
npm run start:api    # optional HTTP harness for public/*.html tests
npm run test:unit
npm run test:app-up  # build + next start GET /
```
