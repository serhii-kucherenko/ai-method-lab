# Screenlane

Comprehensive hiring-screening API + minimal UI.

## Run

```bash
npm install
npm test
npm start
```

Open `http://127.0.0.1:3847/`.

## Domain

Boards (ACL) → jobs + candidates → applications (`applied` → `screening` → `decided`) with weighted scorecards.

## Method

A03 test-first + A10 enterprise gates. See `docs/DEVELOPMENT_WORKFLOW.md` and `docs/FRAMEWORKS.md`.
