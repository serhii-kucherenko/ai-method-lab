# Virulence Predict Studio

Structure and evolution in the score — organism panels, feature-integrated virulence / ARG prediction, contribution views, and sequence-only compare.

## Run

```bash
cd projects/virulence-predict-studio
npm install
npm run dev
```

Open http://localhost:3000

## Tests

```bash
npm test
npm run test:app-up   # live build + next start smoke
```
