# PRD — Grantlane

Grant programs with dual-reviewer activation, milestone budgets, and clawback.

## AC (summary)

Auth; program ACL (admin/officer/reviewer); applications submitted→active→closed; dual sign-off before activate; approved_amount budget ceiling on milestone payments; close gate on milestone resolution; clawback on active grants; audit; HMAC webhooks; outbound notify on activate/close/clawback; pagination; rate limits; UI.
