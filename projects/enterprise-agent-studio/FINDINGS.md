# FINDINGS — Enterprise Agent Studio

## Claim
Multi-agent role orchestration improves enterprise resource plan quality versus a single unchecked agent, especially under high constraints, capacity tightness, and cross-domain links.

## What worked
- Distinct IA (`/domains` `/agents` `/plans`) made the ERP story visible without desk noun-swap.
- Dual-impl goldens locked multi vs single formulas before UI polish.
- Active agent specialization feeds plan inputs so the Agents page is not decorative.

## What to watch
- Soft simulation only — do not overclaim production ERP fidelity.
- Never brand as Agentic ERP; keep honesty fence on landing + `/honesty`.

## Sustain checklist
- ≥20 features, ≥8 pages, ≥30 dual-impl goldens
- `npm run build` + `npm run test:app-up` green
- Guide `docs/guides/41-enterprise-agent-studio-lessons.md`
