# Pocket Molecule Desk

Soft simulation desk for **pocket-conditioned + property-aware molecule plans**, inspired by [arxiv 2607.12349](https://arxiv.org/abs/2607.12349v1). No authors' code shipped with the paper.

**Never brand** ConDitar / msPRL / paOPT / CDH as this product.

## Run

```bash
cd projects/pocket-molecule-desk
npm install
npm run dev          # Next.js UI
npm run start:api    # optional API sidecar
npm test             # unit + app-up live smoke
```

## Unique claim

Pocket-conditioned + property-aware plans (multi-scale pocket conditioning, binding affinity **and** ADMET/developability) versus unconditioned / ligand-only resemblance, affinity-only (no developability), or property-blind pocket fill.

## Honesty

Method experiment — not a ConDitar replacement or commercial SBDD tool. Soft scores only.
