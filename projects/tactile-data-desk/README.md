# Tactile Data Desk

Touch-first chart exploration desk: select → confirm → ask → verify — inspired by paper 2607.14588.

- **Paper:** https://arxiv.org/abs/2607.14588v1
- **Authors' code:** https://github.com/accessible-data-vis/feelogue
- **Guide:** [docs/guides/22-tactile-data-desk-lessons.md](../../docs/guides/22-tactile-data-desk-lessons.md)
- **Design:** [docs/ideas/tactile-data-desk-DESIGN.md](../../docs/ideas/tactile-data-desk-DESIGN.md)
- **Try offline:** open `try.html` in a browser

## Run

```bash
cd projects/tactile-data-desk
npm install
npm test
npm run start:api   # API + static harness on :3849 (or PORT)
npm run dev         # Next.js UI
```

## Claim

Touch-first chart sensemaking with agent for calculation, then verify on the tactile chart — versus speech-only answers that skip tactile grounding.

Honesty: method experiment; never brand as Feelogue, CTDI, or Dot Pad.
