Tactile Data Desk

Select, confirm, ask — then verify on the chart

Score soft layered selection, confirmation, agent calculation, and chart verification against speech-only answers that skip tactile grounding.

Open desk Read honesty

Problem

Speech-only chart answers invent numbers without a tactile select and confirm. Teams need soft-layer grounding, an agent for calculation, and verify-on-chart — not a floating voice reply.

Product

Tactile Data Desk runs a grounded pipeline — select → confirm → ask agent → verify on chart — beside speech-only and other weak baselines, with jobs, audit, and goldens. Soft RTD simulation; no hardware required.

Selling points

  • Compare grounded select → confirm → ask → verify against speech-only answers that skip tactile grounding
  • Keep tactile explore jobs honest with lifecycle, batch siblings, and audit export
  • Browse dual-impl goldens so grounding scores stay checkable
  • Org webhooks and roles so rejects and rotates stay visible

Features

  • Member roles

  • Tactile explore job CRUD
  • Lifecycle transitions
  • Illegal transition reject
  • Optimistic version conflict

  • Audit log + CSV export

  • Offline try page
  • Marketing landing

How it works

  1. Open the desk and create a tactile explore job under a chart topic
  2. Run scenario compare — speech-only vs grounded select-confirm-verify
  3. Advance lifecycle or batch, then export the audit trail

Honesty / limits

Workflow experiment inspired by the paper. Not a replacement for the authors' system. Not a commercial tactile accessibility product. Never brand as Feelogue, CTDI, or Dot Pad.
