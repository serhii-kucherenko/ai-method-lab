# Secure Tutor Studio

Teach secure code with orchestrated tutors. Multi-LLM secure software tutoring workspace inspired by SYNAPSE research — **not** branded as SYNAPSE, **not** an isomorphic lab desk shell.

## Paper
- https://arxiv.org/abs/2607.14601v1
- Authors’ code: none published with this paper

## Run

```bash
cd projects/secure-tutor-studio
npm install
npm run dev
```

Open http://localhost:3000 — marketing landing CTA → `/courses`.

## Test

```bash
npm test
npm run test:app-up
```

## Pages
`/` `/courses` `/tutors` `/lessons` `/compare` `/runs` `/settings` `/honesty`

## Claim
Multi-LLM orchestrated secure tutoring quality vs single-LLM unchecked baselines for secure software education.
