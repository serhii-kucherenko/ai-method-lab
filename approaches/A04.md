# A04 — Plan → execute

```yaml
id: A04

thesis: Planner agent then builder; no code in plan phase
```

## Must

- Written plan (tasks, risks, DoD) merged or accepted before any impl
- Builder follows plan; deviations logged
- Plan phase produces no production code

## Must not

- Coding during planning
- Silent plan abandonment

## Review model

Plan review gate, then code review against plan.

## Team notes

| Size | Compression |
|------|-------------|
| 1 | Same human/AI: strict phase split |
| 2 | A plans; B builds |
| 3+ | Planner role rotates; builders execute |

## Done when

Plan artifact linked; oracle green; deviation log present; metrics recorded.
