## Must

- Research note (options, risks, decision) merged first
- Only then PRD/impl
- Matches simple-papers diverge-then-converge spirit

## Must not

- Code commits before research merge
- Research that is only a rubber-stamp paragraph

## Review model

Research PR reviewed for decision quality; then normal code review.

## Team notes

| Size | Compression |
|------|-------------|
| 1 | Self-gate with checklist |
| 2+ | Separate research reviewer when possible |

## Done when

Research artifact merged first; oracle green; metrics recorded.
