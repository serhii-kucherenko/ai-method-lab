## Must

- Distinct reviewer identity from builder (second model or second person)
- Reviewer checklist: correctness, security, scope creep, missing tests
- Findings must be resolved or explicitly waived before merge

## Must not

- Builder approving own PR as the only review
- Waiving security findings without note

## Review model

Mandatory adversarial pass; human merge still required for wave 1.

| Size | Compression |
|------|-------------|
| 1 | Second AI as adversary; human samples |

| 3+ | Dedicated review lane |
