# A02 — PRD + ERD first

```yaml
id: A02
version: 1
status: active
supersedes: null
thesis: Contracts before code; schema is law
```

## Must

- PRD with testable AC
- ERD (or schema + API contract) approved before implementation starts
- Migrations plan if data model changes
- Impl must not invent tables/fields absent from ERD without ERD revision + note

## Must not

- Code-first schema discovery
- Silent schema drift

## Review model

Review ERD diff before code review. Human or designated architect signs ERD gate.

## Team notes

| Size | Compression |
|------|-------------|
| 1 | AI drafts ERD; human signs before code |
| 2 | B owns ERD; A owns PRD |
| 3+ | Dedicated data/API owner for ERD gate |

## Done when

ERD artifact linked from PR; oracle + schema checks green; metrics recorded.
