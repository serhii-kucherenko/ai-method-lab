# Lessons — Meta Synthesis Studio

**Product:** Meta Synthesis Studio (`projects/meta-synthesis-studio`)  
**Paper:** https://arxiv.org/abs/2607.15247v1  

## What worked

1. **Buyer-first IA** — `/questions` `/searches` `/screens` `/effects` `/analyses` `/compare` sells evidence meta-analysis, not a desk shell.
2. **Dual score clarity** — A keeps screen discipline + heterogeneity; B skips both. Goldens make the gap measurable.
3. **Marketing landing** — Hero brand + one evidence outcome + CTA to questions; honesty fence on `/honesty`.

## What to avoid next time

1. Noun-swapping prior studios (deployments/prompts/policies) into this domain.
2. Claiming live PubMed or publication-ready statistics when the desk is a soft sim.
3. Using `/jobs` `/lifecycle` `/scenario` `/batch` `/goldens` as primary nav.
4. Branding as AutoSynthesis.
