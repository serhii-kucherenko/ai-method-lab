# 36 — Virulence Predict Studio lessons

## Context
Built from SEVA-class virulence / ARG prediction research (structure + evolution vs sequence-only), without branding as the authors' tool or shipping a desk IA clone.

## Lessons
1. **Distinct IA first** — panels / predict / features / compare sells the claim; jobs/lifecycle/goldens does not.
2. **Feature page is the differentiator** — contribution bars make “structure and evolution in the score” visible.
3. **Dual-impl goldens** — keep A/B scoring algebra simple enough to reimplement independently and still match fixtures.
4. **Never brand as paper acronym** — mature display name + honesty Sources keep the Method Lab fence clear.
5. **Live app-up is the sustain gate** — unit green without `next build` + `next start` is not enough.

## Anti-patterns avoided
- Noun-swap desk shell
- Same-tick paper → smoke scaffold
- Shipping authors' trademark as product name
