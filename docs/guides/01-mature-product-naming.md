# Guide 01 — Mature product naming

## Problem

Research ids like `c1592`, `irc6651`, and glue-nouns like `oshamult` leaked into product brands, emails, and UI. That reads like an experiment spreadsheet, not a product.

## Practice

1. Pick a **display name** a stranger understands (2–4 words).
2. Pick a **slug** of ordinary hyphenated words for the folder.
3. Keep statute cites in the *domain* docs — never as the brand.
4. Portfolio, try page, and findings email lead with the display name.

## Refuse

- Shipping UI chrome that only shows a code
- Email subjects full of cell ids
- Renaming a dual-gate clone and calling it innovation

## Score impact

Sustain fails maturity if the public brand violates `docs/PRODUCT_NAMING.md`.
