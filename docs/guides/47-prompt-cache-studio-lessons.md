## What worked

1. **Buyer-first IA** — `/deployments` `/prompts` `/policies` `/costs` `/hits` `/compare` sells LLM cost control, not a desk shell.
2. **Dual score clarity** — A preserves prefix cache + two-tier savings; B rewrites the prefix and loses hits. Goldens make the gap measurable.
3. **Marketing landing** — Hero brand + one cost outcome + CTA to deployments; honesty fence on `/honesty`.
