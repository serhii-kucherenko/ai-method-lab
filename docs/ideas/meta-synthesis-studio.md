# Meta Synthesis Studio

**State:** ready_to_build  
**Slug:** `meta-synthesis-studio`  
**Paper:** https://arxiv.org/abs/2607.15247v1  
**Authors’ code:** none  

## Buyer
Evidence teams running quantitative meta-analyses who need an agentic pipeline from question → search → screen → extract → effect sizes — not a pile of ad-hoc scripts.

## Outcome
A studio to open review questions, craft search strategies, screen literature, extract effects, run pooled analyses with heterogeneity checks, and compare against ad-hoc single-pass baselines.

## Why this paper
Clear multi-agent software claim for automated meta-analysis; stronger comprehensive product than packing/itinerary score-1 peers. No public authors’ code — Method Lab product inspired by the pattern, not a rebrand of AutoSynthesis.
