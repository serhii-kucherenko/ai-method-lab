# DESIGN — Graph RAG Studio

## Brand

**Graph RAG Studio** — hero-level name. Tagline: “Multi-step graphs. Answers with hop trails.”

## Landing brief

- Headline: See every hop behind the answer  
- Support: Build GraphRAG pipelines that extract, consolidate, and retrieve with visible graph evidence.  
- Primary CTA → `/corpora`  
- Selling points: multi-step construction, hop trails, compact profile, scenario proof, auditable runs  
- Honesty: inspired by RAGU; not RAGU/Meno-Lite  

## Visual direction

- Light studio: cool slate + teal accent (not purple AI default, not cream/terracotta)  
- Fonts: "Fraunces" display + "Source Sans 3" body  
- Atmosphere: subtle graph-node pattern / soft radial wash  
- Motion: hero fade-up, hop-trail draw-in, pipeline stage pulse  

## Anti-looks

No purple gradients, no dark-mode-by-default, no pill-stat strips in hero, no isomorphic desk chrome.

## Page compositions

Each route one job; cards only for interactive panels (ask, graph node detail).
