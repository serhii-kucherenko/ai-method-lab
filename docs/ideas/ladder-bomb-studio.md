# Ladder Bomb Studio

**State:** ready_to_build  
**Slug:** `ladder-bomb-studio`  
**Paper:** https://arxiv.org/abs/2607.08417v1  
**Authors’ code:** none  

## Buyer
OT / ICS security engineers who must find ladder logic bombs in IEC 61131-3 PLC programs before they can manipulate actuators or deny operator control.

## Outcome
A studio to register plants, import ladder programs, run formal verification scans that keep function-block bodies in the model, synthesize triggers, review findings, and compare against verifiers that drop FB bodies.

## Why this paper
Score-3 software claim: formal verification + trigger synthesis for ladder logic bombs; next unused pick after Agent Safety Studio. No public authors’ code — Method Lab product inspired by the pattern, not a rebrand of ESBMC-LLB.
