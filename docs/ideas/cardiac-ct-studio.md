# Cardiac CT Studio

**State:** ready_to_build  
**Slug:** `cardiac-ct-studio`  
**Paper:** https://arxiv.org/abs/2607.11287v1  
**Authors’ code:** none  

## Buyer
Cardiac imaging teams who need comprehensive CT segmentation and phenotyping with human-in-the-loop annotation — not auto-only labels that skip expert review.

## Outcome
A studio to manage CT studies, run HITL annotation queues, track multi-structure segments, review phenotypes, apply augmentation policies, and compare against auto-only baselines.

## Why this paper
Clear imaging software claim (HITL + foundation model + phenotyping). Distinct from Alzheimer Predict / Pathology Slide. No public authors’ code — Method Lab product inspired by the pattern, not a rebrand.
