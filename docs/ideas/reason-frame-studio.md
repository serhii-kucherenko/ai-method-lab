# Reason Frame Studio

**State:** ready_to_build  
**Slug:** `reason-frame-studio`  
**Paper:** https://arxiv.org/abs/2607.08403v1  
**Authors’ code:** none  

## Buyer
Teams shipping LLMs into rule-based scientific domains who need multi-agent game-theoretic checks against hallucination — not single-model answers that sound fluent but break logic.

## Outcome
A studio to define scientific rule packs, run multi-agent debate rounds, apply Bayesian/team-game scoring, review hallucination flags, and compare against single-agent baselines.

## Why this paper
Clear software claim (G-Frame adaptive multi-agent + game theory for hallucination). Distinct from Agent Safety Studio (structural monitors). No public authors’ code — Method Lab product inspired by the pattern, not a rebrand of G-Frame.
