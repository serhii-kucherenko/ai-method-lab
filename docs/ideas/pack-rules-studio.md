# Pack Rules Studio

**State:** ready_to_build  
**Slug:** `pack-rules-studio`  
**Paper:** https://arxiv.org/abs/2607.15562v1  
**Authors’ code:** none  

## Buyer
Travel product teams who must generate packing checklists that obey hard safety/luggage rules while still reflecting soft personal preferences.

## Outcome
A studio to manage traveler profiles, hard rules, soft preferences, generate rule-compliant checklists, run preference learning/optimize passes, and compare against preference-only baselines that break rules.

## Why this paper
Clear constrained-personalization software claim (symbolic rules → learning → optimize). Next unused pick after Tactile Chart. No public authors’ code — Method Lab product inspired by the pattern, not a rebrand.
