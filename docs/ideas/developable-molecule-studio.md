# Developable Molecule Studio

**State:** ready_to_build  
**Slug:** `developable-molecule-studio`  
**Paper:** https://arxiv.org/abs/2607.12349v1  
**Authors’ code:** none  

## Buyer
Drug discovery teams who need pocket-conditioned molecules that also pass developability filters — not affinity-only generators that fail later developability checks.

## Outcome
A studio to register binding pockets, run pocket-conditioned generation, apply property-aware optimization, track developability properties, and compare against affinity-only baselines.

## Why this paper
Clear generative chemistry software claim (pocket + developability). Distinct from Disease Drug Studio (disease-aware candidates). No public authors’ code — Method Lab product inspired by conDitar-dev patterns, not a rebrand.
