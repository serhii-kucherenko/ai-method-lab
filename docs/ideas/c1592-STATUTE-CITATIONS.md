## Primary — 19 U.S.C. § 1592

| Subsection | Rule (v0 reading) |
|------------|-------------------|
| **(c)(1)** | Fraudulent violation → civil penalty **not to exceed domestic value** of the merchandise |
| **(c)(2)** | Gross negligence → if duty/tax/fee loss: lesser of domestic value or **4×** loss; if **no** loss: **40%** of dutiable value |
| **(c)(3)** | Negligence → if duty/tax/fee loss: lesser of domestic value or **2×** loss; if **no** loss: **20%** of dutiable value |
| **(c)(4)** | Prior disclosure — separate path; **not** automatic in v0 (`c1592-PD-FENCE.md`) |
