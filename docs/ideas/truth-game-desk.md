## Paper in plain words

Language models often invent confident falsehoods. The paper describes a game-theory-driven multi-agent framework meant to reduce those hallucinations by structuring how agents challenge and score each other’s claims. No public authors’ code shipped with the paper.

**Game-theoretic multi-agent truth plans** (structured challenge/payoff among agents to dampen hallucination) vs **naive baselines** (single-agent unchecked answers; flat majority vote without game structure; confidence-only filters).
