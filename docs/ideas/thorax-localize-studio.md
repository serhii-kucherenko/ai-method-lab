# Thorax Localize Studio

**State:** ready_to_build  
**Slug:** `thorax-localize-studio`  
**Paper:** https://arxiv.org/abs/2607.09305v1  
**Authors’ code:** none  

## Buyer
Chest imaging teams who need disease classification **and** lesion localization on CXR — not classify-only scores without where the finding is.

## Outcome
A studio to manage CXR exams, review multi-disease findings, inspect lesion maps, run localization validation, and compare against classify-only baselines.

## Why this paper
Clear CXR software claim (detect + localize with PCAM). Distinct from Cardiac CT Studio. No public authors’ code — Method Lab product inspired by Inspectra patterns, not a rebrand.
