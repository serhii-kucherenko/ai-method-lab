# Model Compile Studio

**State:** complete  
**Slug:** `model-compile-studio`  
**Paper:** https://arxiv.org/abs/2607.15865v1  
**Authors’ code:** https://github.com/sophgo/tpu-mlir  

## Buyer
ML systems engineers compiling large language models toward efficient accelerator targets who need a multi-pass MLIR-style plan — not a one-shot opaque export.

## Outcome
A studio to register models, configure compile targets, run multi-pass compile plans, inspect artifacts, compare against naive single-pass / target-blind baselines, and audit runs.

## Why this paper
Public code (TPU-MLIR), highest remaining score among code-backed picks, distinct from pathology / drug / virulence / GraphRAG studios.
