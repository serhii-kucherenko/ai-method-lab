# Tactile Chart Studio

**State:** ready_to_build  
**Slug:** `tactile-chart-studio`  
**Paper:** https://arxiv.org/abs/2607.14588v1  
**Authors’ code:** none  

## Buyer
Accessibility / data product teams building chart experiences for blind and low-vision users who need conversational + tactile exploration — not visual-only dashboards.

## Outcome
A studio to register charts, configure layered tactile presentation, define feedback grammar, run conversational explore sessions with select-confirm-ask-verify, and compare against visual-only baselines.

## Why this paper
Concrete accessible multimodal software claim (RTD + LLM conversation) with design patterns builders can ship. Stronger product story than packing/itinerary peers. No public authors’ code — Method Lab product inspired by Graphy patterns, not a rebrand.
