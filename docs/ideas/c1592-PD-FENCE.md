# c1592 — prior-disclosure honesty fence

**v0 must not claim** automatic prior-disclosure (PD) caps under 19 U.S.C. § 1592(c)(4).

## Why

PD eligibility depends on disclosure **before, or without knowledge of, commencement of a formal investigation** — a fact pattern counsel owns. Silent “fix” of investigation-start clocks would be dishonest framing.

## Allowed in digests

- “Optional PD path is an explicit input if we add it later.”
- “v0 forecasts statutory maximums only.”

## Forbidden

- “We detect investigation start and auto-apply PD.”
- Treating PD interest-only negligence caps as the default happy path without dual suite.
